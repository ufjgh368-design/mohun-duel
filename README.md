# 墨魂對決 — 臺灣藝術家知識格鬥 RPG

以答題為武器的回合制知識格鬥遊戲。扮演 12 位臺灣美術先驅,答對題目就能施展以其藝術特色命名的招式,在課堂上寓教於戰。

> 🎨 **姊妹作:[西洋藝術大亂鬥 — Canvas Combat](art-combat/)**
> 同一套戰鬥引擎的西洋藝術史版本,位於 `art-combat/` 子資料夾:34 位藝術英雄(達文西、梵谷、畢卡索、芙烈達……)依「文藝復興 → 巴洛克與浪漫 → 印象與後印象 → 現代藝術革命」分為四大流派,並新增**流派相剋系統**(剋制方傷害 +15%);內建 340 題西洋藝術史題庫(涵蓋記憶、理解、素養、判斷、文化脈絡、趣聞等多元題型)、34 位全員電影級英雄立繪、12 關故事遠征、名作鑑定挑戰、BOSS「空白畫布」、美術館大廳視覺、四樂派專屬戰鬥舞台、古典琶音 BGM。部署到 GitHub Pages 後網址為 `https://<帳號>.github.io/<repo>/art-combat/`,進度與題庫(`cc_*`)與本作(`mh_*`)完全獨立。

**風格**:中國古典武俠 × Dark Fantasy × 現代 Game UI(玻璃擬態、金色光暈、電影級動畫)

## 遊戲特色

- ⚔ **九種模式**:單人(5 級 AI)、雙人同機對戰、練習刷題、生存連戰、BOSS 戰(遺忘之霧)、**闖關模式**(美術史行旅:四大章節 10 關、星級評價、進度保存)、**卡牌模式**(90 張作品卡收集、五張牌組、媒材相剋+答題加成對決、勝利抽新卡)、教師題庫、角色圖鑑
- 🎨 **30 位藝術家**,依創作媒材分為四大門派——**油畫**(陳澄波、廖繼春、李梅樹、顏水龍、楊三郎、李石樵、郭柏川、劉啟祥、陳植棋、張萬傳、洪瑞麟、陳德旺、葉火城、廖德政、沈哲哉)、**膠彩**(郭雪湖、林玉山、陳進、呂鐵州、林之助、陳慧坤)、**水彩**(倪蔣懷、藍蔭鼎、李澤藩、蕭如松)、**雕塑與現代藝術**(黃土水、蒲添生、陳夏雨、楊英風、莊世和)——各有能力值、標籤與三段技能(如「南街殷賑」「水牛群像」「鳳凰來儀」)
- ❓ **內建 210 題**臺灣美術史題庫(三級難度各約 70 題,30 位藝術家人人有題),輕擊出簡單題、重擊出中等題、必殺需連過兩道大師題
- 💥 **戰鬥演出**:衝刺殘影、刀光、粒子爆裂、螢幕震動、慢動作、暴擊、全螢幕必殺動畫
- 🔊 **零素材合成音效**:Web Audio 即時合成古箏五聲 BGM 與全部音效
- 🏆 **結算統計**:答對率、平均時間、Combo、傷害、知識分數、MVP;獲勝解鎖圖鑑作品,成績寫入本機排行榜
- 🎓 **教師模式**:表單建題、CSV 匯入/匯出(相容 Excel / Google 試算表)、可切換「只用自訂題庫」
- 📱 PWA 離線可玩、RWD 響應式、零依賴零建置(純 HTML/CSS/JS)

## 本機執行

任何靜態伺服器皆可,例如:

```
python -m http.server 8123
```

再開 `http://localhost:8123`。(直接雙擊 index.html 也能玩,但 PWA 離線功能需經 http 伺服器)

## 部署到 GitHub Pages

1. 建立 GitHub repo,將本資料夾全部檔案 push 上去
2. repo → Settings → Pages → Source 選 `main` 分支根目錄
3. 完成,網址為 `https://<帳號>.github.io/<repo>/`

## 教師 CSV 題庫格式

欄位順序(教師模式內可下載範本):

```
題目,選項A,選項B,選項C,選項D,答案(A-D),難度(1-3),分類,解說
```

- 答案填 A/B/C/D(對應正確選項)
- 難度:1=輕擊(易)、2=重擊(中)、3=必殺(難)
- 由 Excel / Google 試算表「另存 CSV」即可匯入;支援含逗號與引號的欄位
- 專案根目錄的 [內建題庫.csv](內建題庫.csv) 是全部 210 題內建題的匯出檔(UTF-8 含 BOM),可直接用 Excel 開啟檢閱、修改後再匯入教師模式

## 專案結構

```
index.html        畫面骨架(各 screen 區塊)
css/style.css     全部樣式與動畫
js/data.js        角色資料、題庫、模式與 AI 難度定義
js/audio.js       Web Audio 音效引擎(SFX + BGM)
js/fx.js          背景粒子與戰鬥特效
js/game.js        戰鬥引擎(回合、出題、傷害、CPU AI)
js/ui.js          畫面導航、選角、結算、教師模式、圖鑑、排行榜
manifest.json     PWA 資訊清單
sw.js             Service Worker 離線快取
```

儲存皆使用 LocalStorage:`mh_unlocks`(圖鑑)、`mh_board`(排行榜)、`mh_custom_q`(自訂題庫)、`mh_bgm` / `mh_sfx`(音訊設定)。
